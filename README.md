# swift-calculator

## environment
- [macOS 10.14.6](https://www.apple.com/tw/macos/mojave/)
- [Xcode 10.2.1](https://developer.apple.com/cn/support/xcode/)
- [Swift 4.2](https://swift.org)
- emulator iPhone XR(iOS 12.2)

## 計算機 App
**類似 iPhone 計算機，按鈕方型**
- 基本 AutoLayout 設定
- ViewController 綁定元件
- 加減乘除，連續運算
- 元件對齊
